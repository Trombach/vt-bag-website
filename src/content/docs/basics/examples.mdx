This page lists examples of view transitions, starting with the most basic settings and gradually adding more features. The tables provides a brief description of what makes each example unique and includes a link to the actual demo with its code and detailed description.

## Same-Document View Transition Examples

These examples revolve around a tab component where you can switch between different contents.

<div title="Select a link from the table below">
  <Card title="Same-Document 'Tabs' Example">
    <style>{`
    .tabs {
      pointer-events: none;
    }
    .tab {
      display: inline-block;
      padding: 10px 20px;
      cursor: pointer;
      background-color: #887;
      border: 1px solid #887;
      border-bottom: none;
    }
    .tab.active {
      background-color: #797;
      border-bottom: 1px solid #797;
    }
    .tab-content {
      transform: translateY(-1rem);
      display: none;
      padding: 20px;
      border: 1px solid #888;
    }
    .tab-content.active {
      display: block;
    }
`}</style>
    <div class="tabs" id="sd-example">
      <div class="tab active" data-tab="tab1">
        Dad Jokes
      </div>
      <div class="tab" data-tab="tab2">
        Punny Puns
      </div>
    </div>
    <div class="tab-content active" id="tab1">
      <p>
        Why don't skeletons fight each other?
        <br />
        They don't have the guts.
      </p>
      <p>
        I'm reading a book on anti-gravity.
        <br />
        It's impossible to put down!
      </p>
    </div>
  </Card>
</div>

## Cross-Document View Transition Examples

These examples revolve around an image component where you switch through different images.

<div title="Select a link from the table below">
  <Card title="Cross-Document 'Picture' Example">
    <style>{`
    #image {
      display: block;
      margin: 0 auto;
      mask-image: radial-gradient(
        ellipse at center,
        white 35%,
        transparent 71%
      );
    }
    #images {
      pointer-events: none;
    }
 `}</style>
    <div id="images">

## Image Morph Examples

Morphing thumbnails into hero images is one of the classical use cases for view transition. The transition shows that the small thingy is a placeholder for the bigger one that now expands for you.

{" "}
<div title="Select a link from the table below">
  <Card title="Thumbnail to Detail Morphs">
    <style>{`
#one img{
  width: 30%;
  height: auto;
}
#two img {
  width: 100%;
  height: auto;
}
@media (orientation: landscape) {
  #one {
    width: 40%;
    height: auto;
  }
  #two {
    width: 40%;
    height: auto;
  }
}
`}</style>
    <div style="display: flex; flex-wrap: wrap; align-items: center; justify-content: space-around">
      <div id="one">
        <div class="portrait">![A Portrait](/portrait.webp)</div>
      </div>
      <div id="two">
        <div class="portrait">![A Portrait](/portrait.webp)</div>
      </div>
    </div>
  </Card>
</div>
